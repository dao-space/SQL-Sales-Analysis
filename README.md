# SQL-Sales-Analysis
SQL project focused on exploring, cleaning and analyzing a sales dataset. Includes practical use of JOINs, aggregation, filtering, and business-oriented KPIs. The goal is to extract insights on customer behavior, product performance, and revenue trends.

Objectives

- Identify top-selling products
- Analyze customer purchasing patterns
- Measure monthly revenue trends
- Extract actionable insights for decision-making

Skills demonstrated

- JOIN operations (INNER, LEFT)
- Filtering logic (WHERE, LIKE, IN, BETWEEN)
- Grouping & aggregation (GROUP BY, HAVING)
- Data cleaning and exploration
- KPI computation
